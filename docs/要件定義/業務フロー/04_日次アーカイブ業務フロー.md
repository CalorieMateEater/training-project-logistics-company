# 日次アーカイブ業務フロー

## 1. 目的
Hoge社が完了済みの注文・配送・通知データを日次でアーカイブ保管する業務を整理する。

## 2. 登場アクター
- Hoge社運用担当者
- Hoge社システム
- アーカイブ保管領域

## 3. 業務フロー図
```mermaid
sequenceDiagram
    box rgb(237, 246, 255) 自社
        actor OpsUser as Hoge社運用担当者
        participant HogeSys as Hoge社システム
        participant Archive as アーカイブ保管領域
    end

    OpsUser->>HogeSys: 日次アーカイブ実行確認
    HogeSys->>HogeSys: 完了済みデータ抽出
    HogeSys->>Archive: アーカイブファイル保管
    HogeSys-->>OpsUser: 実行結果通知
```

## 4. 業務の流れ
1. 日次アーカイブBatch が起動する。
2. Hoge社システムが完了済み注文、配送履歴、通知履歴を抽出する。
3. Hoge社システムがアーカイブファイルを生成する。
4. アーカイブ保管領域へ保管する。
5. 実行結果を管理情報へ反映する。
