## 3. 運用ルール
- 主担当が2営業日以上不在となる見込みの場合、第1バックアップへ引継を実施する。
- 第1バックアップも対応不可の場合は、第2バックアップへエスカレーションする。
- 引継時は、未完了タスク、保留課題、連絡先、直近意思決定事項を最低限共有する。
