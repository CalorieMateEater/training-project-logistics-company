# 権限貸与・申請管理簿

## 1. 管理方針
- 本番権限は最小権限とし、原則閲覧系のみ常設する。
- 開発権限、検証権限、本番作業権限は環境ごとに分離する。
- 権限貸与・変更・剥奪は申請番号で追跡する。

## 3. 棚卸ルール
- 毎月第1営業日に権限棚卸を実施する。
- 離任決定時は当日中に権限剥奪申請を起票する。
- 本番一時昇格権限は作業翌営業日までに失効確認する。
