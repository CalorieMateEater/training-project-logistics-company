# 基盤設計概要

## 1. 概要
本書は、Hoge社システムの基盤設計に関する関連資料を整理するための一覧資料である。AWS基盤、通信、実行基盤、データ基盤、セキュリティ、構成管理の設計書を横断的に参照できるようにする。

本システムは `ap-northeast-1` 単一リージョン構成を前提とし、マルチリージョン構成は採用しない。

## 4. 論理名と基盤実装の対応
| 論理コンポーネント | 基盤実装 |
| --- | --- |
| Hoge OrderHub 取込Batch | ECS Fargate Task + HULFT受信完了イベント起動 |
| Hoge OrderHub 配送依頼Worker | ECS Fargate Service |
| 配送連携APIシステム | Private ALB + ECS Fargate Service |
| 配送結果取込Worker | ECS Fargate Service |
| 注文ステータス通知Worker | ECS Fargate Service |
| 日次アーカイブBatch | ECS Fargate Scheduled Task |
| Internal API Gateway | Internal ALB |
| 顧客マスタ管理 | ECS Fargate Service |
| 在庫引当管理 | ECS Fargate Service |
| HULFT受信サーバ | EC2 |
| HULFT送信サーバ | EC2 |
| OrderHub データストア | Amazon RDS for PostgreSQL |
| ファイル着信S3 / 処理済みS3 / 返却ファイル出力S3 / 売上アーカイブS3 | Amazon S3 |
| billing-plan-queue / order-notice-queue.fifo / Bar向け出荷依頼キュー | Amazon SQS |
