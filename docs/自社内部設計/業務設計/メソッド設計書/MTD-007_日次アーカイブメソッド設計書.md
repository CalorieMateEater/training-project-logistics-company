## 3. `archiveCompletedOrders(...)`
### 3.1 シグネチャ
```java
public int archiveCompletedOrders(LocalDateTime threshold)
```

### 3.2 処理概要
1. 指定時刻以前に完了した注文を抽出する。
2. 注文、配送結果、請求・通知履歴をアーカイブ対象へ整形する。
3. アーカイブ出力先へ保存する。
4. 実行履歴を登録し、件数を返却する。
