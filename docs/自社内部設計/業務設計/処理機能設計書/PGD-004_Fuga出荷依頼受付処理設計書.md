# PGD-004 Fuga出荷依頼受付処理設計書

## 1. 基本情報
| 項目 | 内容 |
| --- | --- |
| 処理機能ID | `PGD-004` |
| 関連処理設計書ID | `PDS-007` |
| 論理機能名 | Fuga出荷依頼受付 |
| 概要 | Fuga社からの出荷依頼APIを受け付け、入力検証、顧客確認、在庫引当、出荷依頼待ち登録までを行う。 |

## 2. 入出力
| 区分 | 内容 |
| --- | --- |
| 入力 | Fuga出荷依頼APIリクエスト |
| 出力 | 注文情報、出荷依頼待ち情報、受付応答 |

## 3. 主処理
1. 呼出元識別子とAPI入力値を検証する。
2. 重複依頼を検出する。
3. 顧客確認と在庫引当を実施する。
4. 注文情報と出荷依頼待ち情報を登録する。
5. 現在状態を含む受付応答を返却する。

## 4. 制御方針
- Fuga社の状態照会で参照される `partner_request_id` を一意制御する。
- 営業時間外や配送解放待ちは、受付後に待機状態で保持する。
