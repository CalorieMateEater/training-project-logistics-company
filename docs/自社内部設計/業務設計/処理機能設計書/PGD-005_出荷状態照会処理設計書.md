# PGD-005 出荷状態照会処理設計書

## 1. 基本情報
| 項目 | 内容 |
| --- | --- |
| 処理機能ID | `PGD-005` |
| 関連処理設計書ID | `PDS-008` |
| 論理機能名 | 出荷状態照会 |
| 概要 | 外部連携先からの照会要求に対して、注文状態と配送状態を取得し、利用者区分に応じた条件で応答する。 |

## 2. 入出力
| 区分 | 内容 |
| --- | --- |
| 入力 | 状態照会APIリクエスト |
| 出力 | 出荷状態照会APIレスポンス |

## 3. 主処理
1. 呼出元識別子を判定する。
2. 呼出元ごとの照会キー許可範囲を評価する。
3. 注文情報、出荷依頼情報、最新配送状態を参照する。
4. 状態照会レスポンスを編集する。
5. 照会結果を連携履歴へ記録する。

## 4. 制御方針
- Foo社は `partner_order_id` のみ照会可能とする。
- Fuga社は `partner_order_id` と `partner_request_id` を照会可能とする。
- 対象なしは業務的な未存在として扱い、失敗履歴を残して応答する。
