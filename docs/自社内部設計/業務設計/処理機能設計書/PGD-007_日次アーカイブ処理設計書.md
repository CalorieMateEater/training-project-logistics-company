# PGD-007 日次アーカイブ処理設計書

## 1. 基本情報
| 項目 | 内容 |
| --- | --- |
| 処理機能ID | `PGD-007` |
| 関連処理設計書ID | `PDS-009` |
| 論理機能名 | 日次アーカイブ |
| 概要 | 完了済、取消、例外終了の注文を抽出し、日次アーカイブファイルを生成して実行履歴を記録する。 |

## 2. 入出力
| 区分 | 内容 |
| --- | --- |
| 入力 | 注文情報 |
| 出力 | アーカイブファイル、アーカイブ実行履歴 |

## 3. 主処理
1. アーカイブ対象注文を抽出する。
2. 出力ファイル名を採番する。
3. アーカイブファイルを生成する。
4. 実行結果を実行履歴へ記録する。

## 4. 制御方針
- 対象なしでも実行結果は記録する。
- ファイル出力失敗時も実行履歴に失敗結果を残す。
