# PGD-002 配送会社連携処理設計書

## 1. 基本情報
| 項目 | 内容 |
| --- | --- |
| 処理機能ID | `PGD-002` |
| 関連処理設計書ID | `PDS-003` |
| 論理機能名 | 配送会社連携 |
| 概要 | 送信待ち状態の出荷依頼を評価し、配送解放条件と配送会社別送信制約を満たしたものを Bar社またはFuga社へ送信する。 |

## 2. 入出力
| 区分 | 内容 |
| --- | --- |
| 入力 | 出荷依頼待ち情報、注文情報、明細情報 |
| 出力 | 配送会社送信結果、配送依頼履歴、外部通知履歴 |
| 後続 | Baz請求予定通知、Qux注文通知 |

## 3. 主処理
1. 送信対象の出荷依頼待ち情報を抽出する。
2. 配送解放時刻を評価し、Bar向け案件のみBar社営業時間を評価する。
3. 配送会社向けの出荷依頼電文を編集する。
4. 送信中状態へ更新したうえで配送会社へ送信する。
5. 送信成功時は受付済へ更新し、通知対象を登録する。
6. 送信失敗時は再送待ちへ戻し、次回送信時刻を設定する。

## 4. 制御方針
- Bar向け案件は営業時間外に送信せず、次回営業開始時刻まで待機する。
- 送信中状態を経由し、外部送信中の可視化を行う。
- 一時障害時は即時失敗にせず、再送待ちへ戻す。
