# PGD-003 Foo配送結果返却処理設計書

## 1. 基本情報
| 項目 | 内容 |
| --- | --- |
| 処理機能ID | `PGD-003` |
| 関連処理設計書ID | `PDS-006` |
| 論理機能名 | Foo配送結果返却 |
| 概要 | Foo社向け配送結果返却対象を抽出し、返却ファイルを生成して返却完了へ更新する。 |

## 2. 入出力
| 区分 | 内容 |
| --- | --- |
| 入力 | 配送結果返却通知履歴、注文情報、最新配送状態 |
| 出力 | Foo配送結果返却ファイル、返却完了済の通知履歴 |

## 3. 主処理
1. 返却待ちの配送結果通知履歴を抽出する。
2. 注文情報と最新配送状態を参照する。
3. Foo社向け返却レコードを編集する。
4. 返却ファイルを生成する。
5. 通知履歴を `SENT` へ更新する。

## 4. 制御方針
- 正式な返却手段はファイル返却とする。
- 状態照会APIは補助的な参照経路として扱い、本機能の代替にはしない。
